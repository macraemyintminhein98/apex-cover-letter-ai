# APEX AI Cover Letter Generator

Quickly generate personalized cover letters using AI!

Simply paste your resume and the job description, click 'Generate', and get a tailored cover letter in seconds. Designed for job seekers who need speed and efficiency. 

**Note:** This is a front-end MVP. A full implementation would involve a backend API to interface with OpenAI and a payment gateway like Stripe for processing the $4.99 fee per generation.
